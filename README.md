# react-form

react超级表单

react ==> 用的是es5+react实现的 

查看：
```bash
git clone https://github.com/hechuanhua/react-form.git 
cd react
npm install -g
gulp
```

react-redux  ==> 用react+redux+es6重构了一下

查看：
```bash
git clone https://github.com/hechuanhua/react-form.git 
cd react-redux
npm install
npm run dev（启动webpack-dev-server本地环境）
```

或者  npm run build（webpack本地打包，直接打开build文件夹直接查看index.html） 


生产环境: npm run deploy
